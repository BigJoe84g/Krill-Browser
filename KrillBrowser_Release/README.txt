Krill Browser v2.0 - Standalone Release
=======================================

HOW TO RUN:
-----------
1. Double-click the "start.command" file.
2. If macOS prevents it from running ("Unidentified Developer"):
   - Right-click (or Control-click) "start.command"
   - Select "Open"
   - Click "Open" in the dialog box.

REQUIREMENTS:
-------------
- Java 17 or later must be installed on your computer.
- Internet connection (for loading pages).

CONTENTS:
---------
- KrillBrowser.jar: The main application file (contains all code).
- start.command: Helper script to launch the browser.
